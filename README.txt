# 🌈 MultiTheme Product Store

This is a responsive React + TypeScript application with **three switchable UI themes**. Each theme includes different styles, layouts, and sidebar behavior. Product data is fetched from the [DummyJSON API](https://dummyjson.com/products).

---

## 🔥 Features

- ✅ React + TypeScript
- 🎨 Theme Switcher (3 Unique Themes)
- 🧭 Responsive Sidebar (toggle on mobile)
- 🔍 Product Search Functionality
- 📦 Product Grid (fetched via API)
- ⚡ Deployed on Vercel

- **Styling**: Tailwind CSS
